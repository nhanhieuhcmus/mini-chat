## Run Locally

Clone this project

```bash
  git clone https://github.com/nhanhieuhcmus/mini-chat
```

Go to the root directory

```bash
  cd mini-chat
```

Install dependencies

```bash
  npm i
```

Start the server

```bash
  npm start
```

Start the client

```bash
  cd frontend
  npm i
  npm start
```
